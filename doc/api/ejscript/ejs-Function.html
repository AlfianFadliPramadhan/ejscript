const public bind(thisObj: Object, overwrite: Boolean = true): Void

Bind the value of "this" for the function.

Description
This can set the value of "this" for the function. If overwrite is false, it will only define the value of "this" if it is not already defined.
Parameters
thisObj: Object Value of "this" to define.
overwrite: Boolean If true, overwrite the existing stored value of "this". [default: true]
