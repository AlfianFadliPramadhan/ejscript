Web Page Template.

This parses an web page with embedded Ejscript directives.

The template engine provides embedded Ejscript using <% %> directives. It supports:

<% Begin an ejs directive section containing statements <%= Begin an ejs directive section that contains an expression to evaluate and substitute %> End an ejs directive <%.
