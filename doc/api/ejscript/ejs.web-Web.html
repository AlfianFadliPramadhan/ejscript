Web class manages web applications.

This class initializes the web framework and loads web applications. Apps may be JSGI apps with a *..es extension, template apps with a ".ejs" extension or MVC applications.

static const public load(request: Request): Object

Load a web app.

Description
This routine will load JSGI apps with a ".es" extension, template apps with a ".ejs" extension and MVC applications. This call expects that the request has been routed and that request.route.type is set to the type of the request (es|ejs|mvc).
Parameters
request: Request Request object.
Returns
An exports object with an "app" property representing the application.
