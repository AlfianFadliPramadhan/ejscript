const public addListener(name: Object, callback: Function): Void

Add a listener function for events.

Parameters
name: Object Event name to listen for. The listener will receive events of this event class or any of its subclasses. The name can be a string or an array of event strings.
callback: Function Function to call when the event is received.
const public clearListeners(name: Object = null): Void

Clear listeners for a given event name.

Parameters
name: Object Event name to clear. The name can be a string or an array of event strings. If null, listeners for all event names are cleared. [default: null]
const public emit(name: String, args: Array): Void

Emit an event to the registered listeners.

Parameters
name: String Event name to send to the listeners.
args: Array Args to pass to the listener callback.

Returns

Return the listeners for this emitter.

Parameters
name: String Event name to send to the listeners.
