ejs.db.sqlite Sqlite(options: Object)
Description
Initialize a database connection using the supplied database connection string.
Parameters
options: Object Connection options stipulating how to connect to the database. This can be a simple String or Path specifying the filename for the database, or it can be an object with a "name" property containing an access URI for the database.
Options
nameDatabase name URI specifying the SQLite database to open. Example: file://var/spool/db/database.db.
