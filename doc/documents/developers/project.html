Ejscript Project Resources

The Ejscript project uses a variety of tools and resources to better manage the project, these include source code control, a mailing list and a support forum.

Source Code Repository

The Ejscript source code is hosted in a Git repository. We provide open read-only access. Write access is available to regular contributing developers.

Project Issue Database

The Ejscript open issue and bug list is stored in a GitHub database.

Community Support Forum

For free technical support at our community support forum. This is a community based forum where users can share ideas and ask questions. You can also search the forum for answers to past questions.

Developer Contributors Agreement

Developers are very welcome, both as occasional contributors or core developers. For Embedthis to accept patches or code from any developer, the developer must first have completed the Contributors Agreement. This Agreement is designed to ensure that you are protected and that your patches can be utilized by everyone. All submitted patches that are included in Ejscript are governed by this agreement.

To read more, go to the Ejscript developers page.
