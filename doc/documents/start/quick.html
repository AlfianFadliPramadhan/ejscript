Ejscript Quick Start

  1. Installation

    First install Ejscript on your system. You can download a binary package from http://ejscript.org/downloads/ejs/download.esp and then follow the instructions relevant to your system.

  2. Language Tour

    Take the language tour. This will give you a quick overview of Ejscript and how to use it.

  3. Using the Command Shell

    Read about how to run the command shell to run your programs. Then read how to compile your programs so they can run blazing fast using the Ejscript ejs shell.

  4. Read the doc

    There is a wealth of documentation to read. Try the Language Guide and Programmers Guide first.
