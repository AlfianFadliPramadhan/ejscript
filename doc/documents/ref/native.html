Native APIs

The Ejscript Native APIs consist of C language headers, function prototypes, structure definitions and defines. They are divided into two groups:

Topic Description
Ejscript Ejscript Language (JavaScript) Interpreter
Http Http Core Engine
MPR Platform Runtime Foundation

Stability Classifications

Rather than providing a blanket backwards compatibility guarantee, APIs are classified according to their stability. These Stability Classifications should assist developers in migrating to new versions.
