Reference Guide

The Reference Guide contains documentation describing the Ejscript architecture and the various programming APIs.

Architecture Overviews

This guide provides two architectural overviews:

  • Ejscript Language Architecture
  • Ejscript Web Framework Architecture

Programming APIs

The APIs are divided into two kinds: Script APIs for developers to create Ejscript applications and C APIs for system's developers the enhance and extend the Ejscript language itself by creating native classes and methods.

Group Description
Script Library Ejscript Language Library
Native APIs C Language APIs

Stability Classifications

Rather than providing a blanket backwards compatibility guarantee, APIs are classified according to their stability. These Stability Classifications should assist developers in migrating to new versions of Ejscript.
