Historical Ejscript Roadmap

This roadmap is no longer maintained. Please consult GitHub Ejscript Milestones for a current roadmap.

The Ejscript roadmap is created by listening to user input. As such, it is subject to change. This roadmap is not a commitment to product, but the current thinking about future directions. Those who shout loudest -- often get what they ask for. To request a feature, please send email to: dev@embedthis.com

Ejscript 1.X is now regarded as legacy code. Customers should therefore upgrade to Ejscript 2 as soon as possible.

Please tell us what you would like to see at dev@embedthis.com.
