A complete, integrated JavaScript environment.

### To Read Documentation:

  See http://ejscript.org/products/ejs/doc/product/index.html

### To Build:

    ./configure
    make

Images are built into */bin. The build configuration is saved in */inc/bit.h.

### To Test:

    make test

### To a Script:

    ejs script

### To Install:

    make install

### To Create Packages:

    make package
